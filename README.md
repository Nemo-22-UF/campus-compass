# Campus Compass (C++)

## Description
Campus navigation and pathfinding system designed to help users locate destinations efficiently.

## Features
- Processes campus map data
- Pathfinding between locations
- Structured data handling
- Command-line interaction

## Tech

- Data Structures
- Graph/Pathfinding concepts

## Concepts Used
- Graph traversal
- Data parsing
- Algorithm design
